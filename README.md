# CLASS for w†VCDM

[CLASS](https://github.com/lesgourg/class_public) modified to encompass the
$w_{\dagger}$ VCDM model described in https://doi.org/10.1103/n86r-sjgm.

This code is publicly available at https://github.com/Scherer-M/class_wdaggerVCDM.


#### CLASS version: 3.3.4
